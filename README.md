# Node Js gRPC

Todo demo project.

### Pros
- Fast and compact.
- Single client library, well maintained by CNCF.
- Upload progress feedback.
- Cancel request (HTTP2).
- HTTP2/Protobuf

### Cons
- Requires schema.
- Thick client.
- Still young (less support)
- No native browser support.

### Run
```
node server.js
```

```
node client.js Sample-Title-1 Sample-text-1
```
